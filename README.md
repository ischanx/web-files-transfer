# 轻传 - 网页文件传输工具

一个基于云开发平台实现的**"临时”**网盘，主要功能有：提交和收集文件、~~分享和接收文件~~

## 背景

我想传输或者分享一个文件，不想打开客户端，不想添加新好友，不想登录邮箱。

我想收集文件，不想让聊天客户端刷屏，不想让邮件刷屏，不想一个个手动汇总。

我外出忘记带U盘了，不想安装或者登录客户端，怎么临时存储一个文件呢？
